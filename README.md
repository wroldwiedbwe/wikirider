## Install
Since I use Debian, I can only provide instructions for Debian-based distros.
```bash
# Clone the repo
git clone git@github.com:sadboyzvone/wikirider.git
# Run setup.sh
cd wikirider && ./setup.sh
# Activate virtualenv
source .env/bin/activate
# Run
./wikirun.py
```
